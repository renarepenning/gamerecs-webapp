While video game recommendation systems do exist, such as Quantic Foundry or Steamâ€™s recommendation engine, the few out there lack human feedback and are largely biased toward games produced by large, well-known video game studios. Our goal is to create a video game recommendation system that promotes games developed by independent studios and provides quality recommendations tailored to each individual user.

Problem statement

Our approach to a video game recommendation system seeks to eliminate large-studio bias by placing emphasis on games created by independent studios. Secondly, our recommendation algorithm will incorporate human-generated data as well as a user-generated feedback system. The final product consists of a recommendation algorithm hosted on a website that allows each user to make a profile, obtain and rate video game recommendations, and view previous recommendations. The algorithm inputs 1 game and outputs 5 recommendations.
